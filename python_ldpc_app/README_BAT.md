# Batch файлы для запуска LDPC Application

В проекте есть два batch файла для удобного запуска приложения на Windows:

## run_ldpc.bat

Простой batch файл для быстрого запуска с параметрами по умолчанию.

**Использование:**
```batch
run_ldpc.bat [путь_к_матрице]
```

**Примеры:**
```batch
REM Запуск с матрицей по умолчанию
run_ldpc.bat

REM Запуск с указанием конкретной матрицы
run_ldpc.bat ..\Channel_Codes_Database\BCH_7_4_1_strip.alist.txt
```

**Параметры по умолчанию:**
- Матрица: `..\Channel_Codes_Database\BCH_7_4_1_strip.alist.txt`
- Блоков: 100
- Итераций: 5
- SNR: 0.0 - 5.0 дБ (шаг 0.5)
- Интерливер: none
- Режим канала: 1 (АБГШ)
- Включены: BER и FER

## run_ldpc_advanced.bat

Расширенный batch файл с возможностью настройки параметров.

**Использование:**
1. Откройте файл `run_ldpc_advanced.bat` в текстовом редакторе
2. Измените параметры в секции настроек (строки с `set`)
3. Запустите файл

**Пример настройки параметров:**
```batch
set BLOCKS=50
set INITIAL_SNR=1.0
set END_SNR=4.0
set STEP_SNR=0.5
set INTERLEAVER=random
set MODE=2
```

**Доступные параметры:**
- `MATRIX_FILE` - путь к файлу матрицы
- `BLOCKS` - количество блоков для обработки
- `ITERATIONS` - максимальное количество итераций декодера
- `INITIAL_SNR` - начальное SNR в дБ
- `END_SNR` - конечное SNR в дБ
- `STEP_SNR` - шаг SNR в дБ
- `INTERLEAVER` - тип интерливера (none, regular, random, srandom)
- `MODE` - режим канала (1, 2, 3)
- `INTERFERENCE_SNR` - SNR помехи в дБ (для режимов 2 и 3)
- `P` - параметр p (gamma) для режимов 2 и 3
- `MODULATION` - тип модуляции (1=ФМ-2, 2=ФМ-4)

## Требования

- Windows операционная система
- Python 3.6 или выше установлен и добавлен в PATH
- Файл матрицы должен существовать по указанному пути

## Примечания

- Batch файлы автоматически определяют директорию, где они находятся
- Если Python не найден, будет показано сообщение об ошибке
- После выполнения приложение ожидает нажатия клавиши (pause)
- Для выхода из приложения нажмите любую клавишу
